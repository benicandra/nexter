# Nexter

A modern real estate website built with advanced CSS Grid and SASS.

## Features

- CSS Grid-based layout
- SASS preprocessing with modular architecture
- Responsive design with mobile-first approach
- Clean and modern UI

## Tech Stack

- HTML5
- CSS3
- SASS

## Getting Started

### Prerequisites

- Node.js installed

### Installation

```bash
npm install
```

### Development

```bash
npm start
```

This will run the development server and watch for SASS changes.

### Build

```bash
npm run build:css
```

Compiles, prefixes, and compresses CSS for production.
